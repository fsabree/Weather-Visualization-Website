Project Overview

The purpose of this project is to visualize the weather of 500+ cities across the world of varying distance from the equator, and answer the question, "What's the weather like as we get closer to the equator?" We leveraged data from the OpenWeatherAPI. Once we retrieved the data, we showcased the following relationships using scatter plots. For this project, cities are represented by the latitude. Please see our data page to see full list of cities used in the dataset.

  • Temperature (F) vs. Latitude
  • Humidity (%) vs. Latitude
  • Cloudiness (%) vs. Latitude
  • Wind Speed (mph) vs. Latitude
